FEATURE: Top Vendors Page
BRANCH: feature/top-vendors
STATUS: Complete — merged to master, build passing

WHAT WAS IMPLEMENTED
====================
A full-page analytics view at /analytics/vendors that aggregates expenses by their
description field (used as vendor/merchant name) and ranks them by total spend.
Shows up to the top 20 vendors. Includes a live search input that filters the list
by vendor name.

Each vendor row shows: rank (gold/silver/bronze colouring for top 3), vendor name,
total spend, transaction count, average transaction amount, and a violet progress
bar proportional to the top vendor's spend.

FILES CREATED / MODIFIED
=========================
+ app/analytics/vendors/page.tsx      (135 lines)
    'use client' page component. Reads expenses via useExpenses() hook and computes
    vendor aggregations with useMemo (Map keyed by description, sorted desc by total,
    sliced to top 20). Second useMemo drives search filtering. Renders loading spinner,
    empty state, summary cards, search input, and ranked vendor rows.

    Replaced TrendingUp import with Store. Added
